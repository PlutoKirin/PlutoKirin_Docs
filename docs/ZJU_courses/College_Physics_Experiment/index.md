# 25-26秋冬 大学物理实验（1.5学分）丨xt
## Course Overview
- 评价体系：14个实验（2必做+12选课）60%，期末40%
- 考试形式：闭卷

## Exam Prep
### 14个实验
今年为电子预习报告+实验报告，可以形成一套通用的 Prompt 结合 LLM 来速通，但还是最好从绪论课就开始注意**A类&B类不确定度**的计算，这样期末不会太过紧张（其实难度也不大）。

### 期末
考纲范围为：绪论+示波器+分光计（两个必做实验）

可以结合 CC98 上的整理资料高效复习。

👈同目录下有我的整理版本，如有需要可以自取。

## Reflection on Study 
今日（2026年1月16日）刷98才发现有人会用 LLM 来对实验报告进行查错，帖子下的实验报告大部分都是95+的，实属震惊。这大抵也是一种思维模式的提升。
