我记得我小时候很不喜欢读书，但是后来在初中疫情期间，班主任带我们进行了线上共读《如何阅读一本书》的课程，自那以后我便对课外书，或者说尤其是纸质书籍的阅读有着极强的依附性。

我的高中不允许携带电子设备，会在自习课抓看课外书的同学，晚上按时熄灯，不允许熄灯后在寝室任何的讲话、自习行为。那时我会用小夜灯偷偷在被窝里看书，但是灯太亮会被发现，所以那时的“军备竞赛”就是谁能买到更暗更方便在被窝里看书的小灯。零零散散的阅读时间，我会每晚读30分钟以上，有时是 1h 左右的时间阅读课外书，读到累了就可以安稳睡去。

到了大学，我的纸质书阅读时间很少。但是我发现电子版的书籍阅读真的很没有感觉！包括所有的电子版教材，就是因为如此我不仅不用纸质版教材，也不怎么用电子版教材……

睡前因为我坚持不带手机上床（否则我会一直刷手机），我会翻阅一会儿床头的书籍，该篇以下的所有阅读笔记都是在睡前阅读并折页记录，待到有空时再进行电子版记录与感受撰写的。

我一直认为书籍是一个低成本的体验人生、交流的工具，当作者阐述着与我相近的经历、苦难与想法时，我会忍不住地欢喜，我也喜欢阅读他们对我的建议，不论我用不用的上，我喜欢这种交流。

希望我的阅读笔记也可以与正在阅读着的你产生共鸣，哪怕只有一点点，我觉得都弥足珍贵！
